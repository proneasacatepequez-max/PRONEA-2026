🔌 API Routes (Vercel + Supabase)
Autenticación — tabla: usuarios
POST
/api/auth/login
Verifica correo+contraseña · campos: correo, contrasena_hash, intentos_fallidos, bloqueado_hasta
POST
/api/auth/logout
Invalida token_sesion
POST
/api/auth/recuperar
Genera token_recuperacion + token_recuperacion_expira
Estudiantes — tablas: estudiantes, inscripciones
GET
/api/estudiantes
Lista paginada con filtros: etapa_id, tecnico_id, sede_id, ciclo_escolar, estado
GET
/api/estudiantes/[id]
Detalle completo con inscripciones activas
POST
/api/estudiantes
Crea registro en estudiantes + inscripciones + usuario
PUT
/api/estudiantes/[id]
Actualiza campos de estudiantes (respeta RLS)
Notas — tablas: notas_tareas, notas_examenes, resumen_libro, resumen_etapa
GET
/api/notas/tareas/[inscripcion_id]
Lista notas_tareas de una inscripción (con JOIN tareas_catalogo)
POST
/api/notas/tareas
UPSERT en notas_tareas · campos: inscripcion_id, tarea_id, nota (0–5)
POST
/api/notas/examenes
UPSERT en notas_examenes · campo: nota_original (0–100). puntos_obtenidos es GENERATED
POST
/api/notas/calcular/[inscripcion_id]
Recalcula y actualiza resumen_libro y resumen_etapa
SIREEX — tabla: resumen_etapa
GET
/api/sireex/listos
WHERE estado='listo_validar' AND validado_digeex=false
POST
/api/sireex/validar
SET validado_digeex=true, validado_por, validado_en
GET
/api/sireex/exportar.csv
Genera CSV con campos requeridos por SIREEX
Configuración pública — tablas: avisos, acuerdos_ministeriales, configuracion
GET
/api/public/info-login
Retorna: avisos activos, acuerdos vigentes. No requiere auth.
GET
/api/configuracion
Parámetros del sistema (solo lectura para no-admin)
PUT
/api/configuracion/[parametro]
Actualiza valor en configuracion.valor (solo administrador)
